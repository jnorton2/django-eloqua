django-eloqua
=============

Eloqua implementation for Django, WIP!

Requirements
============
django >= 1.4
requests >= X

Installation
============

$ pip install django-eloqua

Setup
=====

ELOQUA_SITE_ = 'ACME.Inc'
ELOQUA_USERNAME = 'Foobar'
ELOQUA_PASSWORD = 'f00b4r'
